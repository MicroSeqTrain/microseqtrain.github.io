In GalaxyTrakr, clicking on the Workflow button in the site's top navigation window will take you to your saved workflows. In order to save a workflow to this easy-to-access location:

  • Click the Shared Data button in the top navigation menu, and select Workflows. This will take you to the list of all published workflows that have been imported and shared publicly on GalaxyTrakr. You will be able to see workflows' names, owners, annotations, community tags and ratings, and date of last update.
  • Search for either a particular workflow name, if you know the workflow you would like to access, or other attributes of workflows described in the previous item that may help you narrow down your search.
  • Once you have found a workflow you would like to use, click on the dropdown menu next to its name. To run it once, select Run. To save the workflow to your saved workflows, click Import. To download the workflow as a .ga file, select Save as file.
  • Click on the Workflow menu item in the GalaxyTrakr navigation header to ensure the workflow has been saved to this list. You can click the arrow that looks like a Play button to run the workflow from this list.
