About Us

SeqSero2 is a tool used to predict the serotype of Salmonella isolates.

Coming soon!

You may begin with a paired collection of forward and reverse FASTQ files, two datasets containing forward and reverse FASTQ files, or FASTA files. For assistance with uploading data, please see the GalaxyTrakr Basics tutorials.

Once you have your data in your history, search for SeqSero2 in the search tools bar and select the SeqSero2 tool.

Select your chosen dataset type from the dropdown of the tool. If you have a dataset of the type selected, it will autopopulate in the next dropdown. If this is not the input you would like to use, use the dropdown to select the correct dataset.

For Analysis mode, allele mode was what was used to evaluate SeqSero2 by the GenomeTrakr team. Please contact us if you would like to discuss whether k-mer mode would be better for you.

Leave the dropdown for Algorithms for BWA mapping at the default, which is mem. This is the only algorithm for which SeqSero2 is currently optimized.

Once you are happy with the inputs, click Execute to run the tool.

SeqSero2 will create an output containing a table for each sequence uploaded detailing the O and H antigens coded for, the predicted identification of the Salmonella species, the predicted antigenic profile, the predicted serotype, potential inter-serotype contamination, if any, and any notes about the sequence, including whether the sequence uploaded does not align with Salmonella genomes.

Documentation for SeqSero2 can be found here on the developer's GitHub page.
