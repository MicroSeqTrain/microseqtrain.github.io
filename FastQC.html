About Us

FastQC is a GalaxyTrakr tool that will take fastq, bam, or sam input and output data on the quality of that read.

Coming soon!

For the purposes of this tutorial, it is assumed that the user has previously created a history, uploaded fastq or fasta files, and created a paired list of them in GalaxyTrakr. If you have not already done so and would like to learn more about those steps, please visit this tutorial for fastq import and this tutorial for creating a collection of fastq pairs.

  1. While in the history where you have placed your fastq files, search for FastQC in the Tools search bar in the left pane of the GalaxyTrakr main page. Select the matching tool.

    GalaxyTrakr will attempt to autopopulate the list of files to run the tool on from the current history. Use the icons for single or multiple fastq files and the dropdowns to select your data sources.

    For the purposes of foodborne illness surveillance, tuning the default options of the tool is not generally advisable. If you have questions about uses of contaminant lists or other options on the FastQC tool page, please contact us.

  2. If you would like an email notification when the tool completes working, toggle the Email notification switch.
  3. Click Execute when you are satisfied with your choices to trigger the tool.
  4. The outputs of the tool can be viewed at two levels:
    • At the raw data level, the quality of each individual forward and reverse read in the browser (by clicking on the eye next to each read's name) or by downloading, by clicking on the name of the read and then clicking on the floppy disk button that appears. This file will be in txt format.
    • Alternatively, a webpage report containing visualizations of the raw data can be found by clicking on #:FastQC on stepname: Web page and navagativing to the level of each individual read, clicking on the read name, and clicking on the eye icon beside it. That web page will include the following:
      • Basic Statistics
      • Per base sequence quality
      • Per sequence quality scores
      • Per base sequence content
      • Per base GC content
      • Per sequence GC content
      • Per base N content
      • Sequence Length Distribution
      • Sequence Duplication Levels
      • Overrepresented sequences
      • Kmer Content
