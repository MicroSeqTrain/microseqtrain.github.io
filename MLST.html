About Us

MLST is a tool that will run input genomes against PubMLST schemes and output the scheme most closely matching the input genome along with the alleles used to select the scheme.

Coming soon!

MLST can be used with FASTA files. If your sequences are not already assembled and you would like assistance in doing so, please see the skesa tutorial.

  1. Once you have your data in your history, search for MLST in the search tools bar and select the MLST tool.
  2. Input either the multiple datasets containing your FASTA files or a collection of such files.
  3. If you do not desire to specify advanced parameters, click Execute to run the tool.
  4. The output will be a tabular file that lists the filenames of your input files, the scheme found to match each input file, and the alleles used to match the scheme to the input.

Documentation for MLST can be found at these locations:
